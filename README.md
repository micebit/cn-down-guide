# cn-down-guide

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存放，内容涵盖信息整理、操作指引或工具说明等方向。所有页面均为独立文件，不依赖后端服务，可直接通过浏览器打开或托管于静态页面服务。

## 目录说明

```
cn-down-guide/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的公共资源（如图片、样式、脚本）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件为一个独立页面，文件名通常反映其内容主题。
- `assets/` 目录存放页面共用的静态资源，部分页面可能自带内联资源而不引用此目录。

## 页面归档说明

- 所有页面均为纯静态 HTML，兼容主流现代浏览器。
- 页面内容可能涉及信息整理、步骤说明或工具展示，不包含动态数据请求。
- 每个页面都经过基础可读性与结构检查，但未做大规模兼容性测试。
- 页面以时间或功能分类命名，便于查找与引用。

## 维护说明

- 本仓库不定期更新，新增页面或对已有页面进行修正。
- 欢迎提交 Issue 或 Pull Request 以修正内容错误或补充有用页面。
- 提交前请确保页面为独立 HTML，无外部依赖（公共资源除外），且内容符合仓库主题。
- 本仓库不针对任何特定域名或网站，仅作为通用 HTML 页面归档使用。

## 使用方式

1. 克隆或下载本仓库。
2. 直接打开 `pages/` 下任意 HTML 文件即可浏览对应页面。
3. 也可将整个仓库部署到任意静态托管服务，通过链接访问各页面。
